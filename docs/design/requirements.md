**作成日:** 2026-06-07

## 1. 機能要件

### 1.1 ページ一覧

| ページ | URL | 主要コンテンツ |
|--------|-----|----------------|
| トップ | `/` | 最新お知らせ・クイックリンク・直近の予定・緊急連絡先 |
| お知らせ | `/news` | カテゴリーフィルター付きお知らせ一覧 |
| ゴミ収集 | `/garbage` | 収集日カレンダー・分別ガイド・粗大ゴミ案内 |
| 防災情報 | `/disaster` | 緊急連絡先・避難場所一覧・日頃からの備え・関連リンク |
| 活動紹介 | `/activities` | 主な活動・年間行事カレンダー・役員紹介 |
| お問い合わせ | `/contact` | 連絡方法案内・問い合わせフォーム（メーラー連携） |

### 1.2 機能詳細

#### お知らせ機能（`/news`）

- カテゴリー分類：イベント / お知らせ / 重要 / ゴミ
- カテゴリーフィルタータブ（クライアントサイドでの絞り込み）
- 各お知らせに日付・カテゴリーバッジ・タイトル・本文を表示

#### ゴミ収集情報（`/garbage`）

- 曜日ごとの収集種別を一覧表示（月〜土）
- 分別ガイド：燃やすゴミ・缶びんペットボトル・プラスチック・古紙古布
- 粗大ゴミの申し込み方法と横浜市サイトへの外部リンク
- ゴミステーション案内

#### 防災情報（`/disaster`）

- 緊急連絡先：警察・消防救急・栄区役所・栄消防署・横浜市災害情報・町内会長
- 避難場所一覧：笠間小学校・笠間中学校・笠間公園・笠間公民館（種別・用途付き）
- 日頃からの備え：非常持ち出し袋・家の安全対策・家族での取り決め
- 関連リンク：横浜市ハザードマップ等

#### お問い合わせフォーム（`/contact`）

- 入力項目：お名前（必須）・メールアドレス（必須）・種別（選択）・内容（必須）
- 種別：入会について / 活動・イベントについて / ゴミ収集について / 防災について / ご意見・ご要望 / その他
- 送信方式：`mailto:` リンクを使ったメーラー起動（サーバーレス）
- 送信後に確認メッセージを表示

## 2. 非機能要件

### 2.1 レスポンシブデザイン

- ブレークポイント：sm（640px）/ md（768px）/ lg（1024px）
- モバイルファースト設計
- タッチ操作でボタン・リンクが押しやすいサイズ（最小44px）

### 2.2 アクセシビリティ

- セマンティックHTML（`<header>`, `<main>`, `<section>`, `<article>` 等の適切な使用）
- 画像に `alt` 属性を設定
- フォームの `<label>` と `<input>` の関連付け
- カラーコントラスト比 WCAG AA 準拠

### 2.3 SEO・メタ情報

- 各ページに `<title>` と `<meta name="description">` を設定
- OGP タグ（og:title, og:description, og:type）を設定

### 2.4 パフォーマンス

- Next.js の SSG（静的サイト生成）を活用
- 画像は Next.js `Image` コンポーネントを使用（使用箇所）

### 2.5 セキュリティ

- 外部リンクには `rel="noopener noreferrer"` を付与
- お問い合わせの個人情報はサーバーに送信せずメーラーに渡すのみ

---

## 3. 制約・仮定

- コンテンツデータは各ページファイル内の静的配列として管理（CMSなし）
- お問い合わせはメーラー経由のため、送信成否の確認はできない
- 役員の個人名はプライバシー保護のため掲載しない（活動紹介ページ）
